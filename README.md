## Assignment 2

This repository contains source code, screenshots, and supporting materials for the
Assignment 2: Syntax, Semantics, and Memory Management

## Assignment Overview
The purpose of this assignment is to analyze syntax, semantics, type systems, and
memory management across multiple programming languages.

The assignment is divided into two main parts:
- Part 1: Syntax and Semantics (Python, JavaScript, C++)
- Part 2: Memory Management (C++, Java, Rust)

## Repository Contents
- Source code files for all programming examples
- Screenshots showing program outputs and error messages
- Examples demonstrating:
  - Syntax errors
  - Type system behavior
  - Memory management techniques
